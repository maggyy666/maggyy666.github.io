GoodFoodMVC

MVC Fullstack application, JQuery, HTML5, Bootstrap, MongoDB

GoodFoodMVC is a project based on the ASP .NET MVC framework. The project consists of mechanisms for adding, editing and deleting recipes. Mechanism-based login restricts non-logged-in users from accessing certain features.When it comes to the database storing recipe and user data, MongoDb was used in this project.

MLVehiclePricePredictor

Machine Learning application, Selenium, Pandas, Scikit-learn, Joblib

Python-based web scraper designed to extract vehicle data from the popular vehicle marketplace OTOMOTO. The extracted dataset is then preprocessed and prepared for machine learning (ML) analysis. The project utilizes a random forest ML model to predict optimal pricing for vehicles based on their features.

WPF Desktop Application, XAML, JSON, C#

WPF Application designed to facilitate railway ticket booking for users. This intuitive and user-friendly application allows passengers to search for available train routes, view ticket prices, and book tickets seamlessly.
